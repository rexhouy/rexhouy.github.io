荣耀6解锁

Open usb debug in developer mode

Test if the phone has connected by using following command

$ adb devices

Go to fastboot mode

$ adb reboot-bootloader

Test if it is in fastboot mode

$ fastboot devices

Unlock

$ fastboot oem unlock ****************

"*************" is unlock code. Get this code from vendor.

Test if it has been unlocked

$ fastboot oem get-bootinfo

Restart

$ fastboot reboot
