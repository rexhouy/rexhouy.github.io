2 bundle

manage application dependencies

2.1 install

gem install bundle

2.2 usage

put Gemfile in project root directory. file name is case sensitive.

example Gemfile 

source 'https://rubygems.org'
gem 'nokogiri'
gem 'rack', '~>1.1'
gem 'rspec', :require => 'spec'

install gems specified in gemfile 

bundle install
